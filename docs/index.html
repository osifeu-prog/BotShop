איך מרוויחים ב-SLHNET?

SLHNET נבנית כשרשרת אנושית: כל אחד מקבל חנות דיגיטלית, פרופיל לשיתוף ותמריץ להפיץ את האקו-סיסטם.

  • יוצרים פרופיל עסקי + חנות דיגיטלית דרך הבוט.
  • מקבלים לינקים ייחודיים לשיתוף החנות והפרופיל.
  • כל שיתוף מייצר תנועה לרשת ואוסף נקודות פעילות.
  • אחרי 39 שיתופים איכותיים או תשלום של 39 נפתחת גישה מלאה.
  • מחזיקי SLH על BSC מקבלים גישה מלאה חינם לאחר אימות ארנק בבוט.
מודל חברתי-פיננסי

המטרה: לחבר עסק אמיתי, נכס דיגיטלי וטוקן בתוך רשת שקופה, בה כל אחד יודע מאיפה הגיע הערך ומי תרם לצמיחה.

מנוע הריפרל המדורג ייבנה בשכבות החל מבוט הגייטוויי ועד למודול DAO בעתיד.

תשלום 39 וגישה מלאה

הצעד הראשון הוא הצטרפות לקהילת העסקים הסגורה. ניתן להצטרף באמצעות:

  • תשלום בנקאי / PayBox / Bit / PayPal / TON לפי ההוראות בבוט.
  • או פעילות של 39 שיתופים איכותיים, כפי שיוגדרו בבוט בהמשך.

אחרי אישור התשלום, תקבל קישור לקבוצת העסקים + קבלת מעקב אישי ברשת ההפניות.

איך מתחילים?

  1. נכנסים לבוט בטלגרם: Buy_My_Shop.
  2. שולחים /start ומקבלים את שער הכניסה.
  3. מבצעים תשלום דרך אחת האופציות שיופיעו.
  4. מקבלים אישור ידני + קישור לקהילה.

בעתיד מנגנון אישור ותיעוד ישולב ישירות במודולי ה-API ובלוח ניהול המשקיעים.

בורסת SLHNET (טיוטה ראשונית)

המטרה: לאפשר מסחר P2P בטוקן SLH, טוקני קהילה ו-NFTים בצורה שקופה ופשוטה, ישירות מתוך הרשת. שכבת ה-API כבר נבנית ותחובר למסך זה בשלבים הבאים. 

נתוני עסקאות יופיעו כאן ברגע שה-API יתחיל לאסוף אותן.

פאנל ניהול / API למשקיעים

שכבת ה-API של SLHNET כבר כוללת נקודות קצה לריפרלים, פיד ותצפית על פעילות. פאנל ניהול גרפי למשקיעים ואדמינים יישב מעל ה-API הזה. 

GET /health
GET /api/token/price
GET /api/token/sales?limit=50
GET /api/posts?limit=20
GET /api/referral/stats
GET /api/referral/tree/<user_id>
POST /api/referral/track_visit
SLHNET כל הזכויות שמורות נבנה כפרויקט אקו-סיסטם קהילתי סביב טוקן SLH.
