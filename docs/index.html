פאנל ניהול / API למשקיעים

שכבת ה-API של SLHNET כבר מתחילה להיבנות כמרכז ניהול נתונים: ריפרלים, פיד, פעילות וסטטוס משתמשים. מעליה יישב פאנל ניהול גרפי למשקיעים ואדמינים.

נקודות קצה מתוכננות:

  • GET /health סטטוס שרת.
  • GET /api/token/price מחיר SLH בייצוג לש"ח.
  • GET /api/token/sales?limit=50 עסקאות אחרונות.
  • GET /api/posts?limit=20 פוסטים אחרונים.
  • GET /api/referral/stats סטטיסטיקות ריפרל.
  • GET /api/referral/tree/<user_id> עץ הפניות למשתמש.
  • POST /api/referral/track_visit מעקב אחרי כניסות דרך לינקים.
